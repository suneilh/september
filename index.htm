Introduction

HTML (HyperText Markup Language) is the language of the web. HTML codes are used to display web pages which may contain formatting (e.g. boldface, underline), links (such as hyperlinks or email links), or embedded multimedia (e.g. images, icons, video audio).

Some examples of HTML codes are:

<head>, <title>, <body>, <a href = "External Link">, <br>, <table>

These codes are arranged in the document to produce formatted web pages. The image below shows the generated web page on top which was a result of HTML code on the bottom. Try to match each line of HTML code in the image below to see how the text is rendered on the top image



As you can see, HTML coding can get complicated. Fortunately, there are many programs that have been developed which simplify creating web pages. But first, it is also important to understand the difference between Web Development and Web Design.


Difference between Web Development and Web Design

Web Developers are programmers who create websites by using code and/or software tools based on specifications given to them. Some of the tools used by web developers include programming languages such as JavaScript, CSS, Python, PHP etc. A strong knowledge of computer science and/or software development is needed to be a good web designer.

Web Designers are not programmers. They can develop websites by using WYSIWYG (What you see is what you get) tools. These tools are visually appealing, easy to use, and use drag and drop elements to create websites. Examples of these tools are Wordpress, Google Sites, Weebly, and Wix. We will be using these tools to create websites in different content areas.

In this course, as a web designer/project manager, you will assume various roles when completing your web projects. Some of these roles include project design, setting specifications, identifying users' need, designing user experience/interface, developing content, graphic design, search engine optimization (SEO), testing/evaluation of web sites, and web publishing.

Many entrepreneurs use blogs and e-commerce platform with graphical user interface to easily establish a web presence for their business.
